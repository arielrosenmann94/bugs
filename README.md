# 🐛 Pokédex Bugueada — Práctica de Debugging

## 📖 ¿Qué es esto?

Este proyecto es una **práctica de debugging** diseñada para estudiantes del curso **Desarrollo de Aplicaciones con Django y Python**.

Encontrarás una aplicación Django completa — una Pokédex — que tiene **50 errores intencionales** escondidos en su código. Tu trabajo es hacerla funcionar desde cero usando tus conocimientos de Django y la IA como copiloto.

---

## 🎯 Objetivo

La aplicación **no funciona**. Ni siquiera arranca.

Tu misión es:
1. Levantar el servidor
2. Corregir los errores uno por uno
3. Lograr que **todas las funcionalidades** operen correctamente

---

## 🧠 ¿Qué vas a practicar?

- Leer y entender **tracebacks** de Django
- Depurar errores en **modelos, vistas, URLs, formularios, templates y configuraciones**
- Usar la **IA como copiloto** para debugging (no como solucionador automático)
- Aplicar buenas prácticas de **resolución de problemas**

---

## ⏱️ Duración estimada

**2 horas 30 minutos a 3 horas**, dependiendo de tu nivel.

---

## 🚀 ¿Cómo empiezo?

Lee el `README.md` dentro de la carpeta `pokedex_project/` para las instrucciones de instalación y configuración.

```bash
cd pokedex_project
cat README.md
```
